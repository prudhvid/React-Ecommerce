E-Commerce Website
===

Filter through items from own API, add them to your cart, and see similar items.

![demo](ecommerce.gif)

Instructions

steps to run the website

1. install pm2

	`npm install pm2 -g`


2. install packages

	`npm install`


3. Build the files
	`npm run build:prod`


4. Start pm2 process
	`pm2 run prod.json`

5. open the site on http://localhost:4040
